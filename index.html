llama.cpp launcher

Интерактивная сборка команд для setup, download и run

Сначала один раз клонируй репозиторий скриптов по HTTP в папку `scripts`, затем выбери GPU-профиль, вставь ссылку на GGUF и запускай готовые команды.

1. Профиль GPU

Базовые значения берутся из `profiles.sh`. Количество GPU автоматически перестраивает команды под multi-GPU.

2. Модель Hugging Face

Вставь ссылку на GGUF-файл, и поля модели заполнятся автоматически.

3. Runtime

Быстрые настройки для команды запуска.

4. Secrets

Секреты подставляются только в соответствующие команды.
